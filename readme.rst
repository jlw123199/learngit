Source Code grammar
-----
This is a normal text paragraph. The next paragraph is a code sample::

   function(){
      alert("this code chunk.");
   }

This is a normal text paragraph again.

Out link grammar
-----
This is a paragraph that contains `a link`_.

.. _a link: http://www.baidu.com/


Chapter grammar
-----
: # 有上标线, 用以部分 |today|
: * 有上标线, 用以章节
: =, 用以小节
: -, 用以子节
: ^, 用以子节的子节
: ", 用以段落
